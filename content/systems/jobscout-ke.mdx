## Overview

A tool that aggregates job listings from multiple Kenyan job boards, filters them based on user preferences, and sends real-time notifications via Telegram.

## System Architecture

- **Scraping:** Playwright for dynamic sites, REST APIs where available
- **Aggregation:** Python backend with SQLite storage
- **Filtering:** Keyword matching and skill extraction
- **LLM:** Groq LLaMA for summarising job descriptions
- **Notifications:** Telegram bot with inline buttons

## Results

- Covers 7 major job platforms
- 500+ users in beta
- Average time to apply reduced by 70%
