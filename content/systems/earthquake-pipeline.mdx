## Overview

A real-time data pipeline that ingests earthquake data from the USGS API, transforms it, and loads it into PostgreSQL. The data is visualized in Grafana dashboards for seismic activity monitoring.

## Architecture

- **Ingestion:** Python script fetching USGS GeoJSON every 5 minutes
- **Orchestration:** Apache Airflow DAG for scheduling and monitoring
- **Storage:** PostgreSQL with time-series optimization
- **Visualization:** Grafana dashboards with maps and time-series charts

## Key Metrics

- Processes 100+ events daily
- Historical data since 2000 (1M+ records)
- Dashboard used by geology students and researchers
