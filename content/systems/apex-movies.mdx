## Overview

A responsive web app that lets users search and discover movies using The Movie Database (TMDB) API. It features infinite scroll, filtering by genre, and detailed movie pages.

## Features

- Search movies by title
- Filter by genre, year, rating
- Infinite scroll pagination
- Movie details with cast and trailer links
- Dark/light theme toggle

## Implementation

- Built with React functional components and hooks
- State management with React Query for caching
- Styled with Tailwind CSS
