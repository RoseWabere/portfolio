## Overview

A compliance dashboard for a Kenyan bank that automates the monitoring of regulatory requirements (e.g., CBK guidelines). It uses dbt for data transformations and Power BI for visualization.

## Architecture

- **Data Warehouse:** PostgreSQL with star schema
- **Transformations:** dbt models for compliance metrics
- **Orchestration:** Apache Airflow for daily refreshes
- **Visualization:** Power BI with row-level security

## Key Features

- Real-time alerts for non-compliance (email/Slack)
- Drill-down reports by branch, product, and officer
- Historical trend analysis

## Results

- Reduced compliance reporting time from 40 hours/month to 2 hours/month
- Enabled proactive risk management
