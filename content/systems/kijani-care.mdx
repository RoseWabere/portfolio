## Overview

Kijani Care 360 is an environmental monitoring platform that tracks tree planting initiatives across Kenya. It ingests data from field teams, processes it through a geospatial ETL pipeline, and visualizes it in real-time dashboards.

## Architecture

- **Data Sources:** Mobile forms, CSV uploads, satellite imagery
- **ETL:** Python scripts with PostGIS geospatial transformations
- **Database:** PostgreSQL/PostGIS for spatial queries
- **Backend:** FastAPI REST API with GeoJSON support
- **Frontend:** React with Mapbox GL for interactive maps

## Impact

- 10,000+ records monitored
- Real-time updates for NGOs and government agencies
- 80% reduction in manual reporting time
